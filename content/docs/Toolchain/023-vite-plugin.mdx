---
title: 常见的 Vite Plugins 有哪些？
---

Vite 拥有丰富的插件生态，以下是一些**常见且实用的 Vite 插件**，按功能分类列出，涵盖开发、构建优化、框架集成、体验增强等多个方面：

---

### 🔧 核心功能增强类

| 插件名                        | 描述                             |
| -------------------------- | ------------------------------ |
| **@vitejs/plugin-vue**     | Vue 3 官方支持插件，处理 `.vue` 文件      |
| **@vitejs/plugin-vue-jsx** | 支持 Vue 中的 JSX/TSX              |
| **@vitejs/plugin-react**   | 支持 React 的插件，内置支持 Fast Refresh |
| **@vitejs/plugin-legacy**  | 支持旧版浏览器（如 IE11），会自动注入 polyfill |
| **vite-plugin-svelte**     | 支持 Svelte 框架                   |
| **vite-plugin-solid**      | 支持 SolidJS 框架                  |

---

### ⚙️ 构建优化与能力增强

| 插件名                          | 描述                                 |
| ---------------------------- | ---------------------------------- |
| **vite-plugin-compression**  | 支持构建时自动压缩资源（gzip / brotli）         |
| **rollup-plugin-visualizer** | 构建后生成依赖体积可视化报告                     |
| **vite-plugin-inspect**      | 可视化调试 Vite 插件行为（查看中间插件输出）          |
| **vite-plugin-dts**          | 自动生成 TypeScript 的 `.d.ts` 类型声明文件   |
| **vite-plugin-esbuild**      | 替换默认的 Babel/TS 编译，使用更快的 esbuild    |
| **vite-plugin-eslint**       | 在开发时启用 ESLint 报错提示                 |
| **vite-plugin-style-import** | 支持按需引入样式，例如 `antd`, `element-plus` |

---

### 💡 开发体验提升

| 插件名                         | 描述                                        |
| --------------------------- | ----------------------------------------- |
| **vite-plugin-pages**       | 基于文件系统自动生成路由（适用于 Vue/React）               |
| **vite-plugin-layouts**     | 配合 vite-plugin-pages，支持布局组件自动注入           |
| **vite-plugin-auto-import** | 自动引入 API（如 `ref`, `reactive`, `useRoute`） |
| **unplugin-vue-components** | 自动按需注册组件，无需手动`import`                    |
| **vite-plugin-mdx**         | 支持 MDX（Markdown + JSX）                    |
| **vite-plugin-md**          | 直接在项目中使用 `.md` 文件（渲染成 Vue 组件）             |

---

### 🌐 网络与服务代理

| 插件名                   | 描述                         |
| --------------------- | -------------------------- |
| **vite-plugin-mock**  | 本地提供 mock 接口，便于前端联调开发      |
| **vite-plugin-proxy** | 更灵活的 dev server proxy 配置方式 |

---

### 🌈 UI 与样式

| 插件名                             | 描述                                   |
| ------------------------------- | ------------------------------------ |
| **unocss / tailwindcss plugin** | 支持原子 CSS（如 UnoCSS、WindiCSS、Tailwind） |
| **vite-plugin-svg-icons**       | 支持 SVG 图标打包为 sprite                  |
| **vite-plugin-purge-icons**     | 图标按需自动导入与使用（结合 Iconify）              |

---

### 🎮 示例插件（官方提供）

| 插件名                            | 描述                                |
| ------------------------------ | --------------------------------- |
| **vite-plugin-pwa**            | 开启渐进式 Web 应用（PWA）支持               |
| **vite-plugin-html**           | 自定义 HTML 模板内容与变量注入                |
| **vite-plugin-env-compatible** | 支持 `.env` 中的变量自动注入（兼容 webpack 风格） |

---

### 🧩 多功能整合类

| 插件名                                                        | 描述                                        |
| ---------------------------------------------------------- | ----------------------------------------- |
| **unplugin系列**（如 `unplugin-auto-import`, `unplugin-icons`） | 支持跨框架使用的插件系列，适配 Vite/Rollup/Webpack 等构建工具 |
