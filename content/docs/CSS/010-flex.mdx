title: 讲讲 CSS Flex，以及应用场景？
---

CSS Flex（全称：Flexible Box Layout）是一种**一维布局模型**，用于在容器中高效地分配空间并对齐元素。它可以让你方便地创建水平或垂直方向的对齐布局，尤其适用于在不同屏幕尺寸下的自适应设计。

---

## 一、Flex 基本概念

### 1. 启用 Flex 布局

要使用 Flex，只需给父容器设置：

```css
display: flex;
```

此时，容器内的直接子元素就变成了 “flex item”。

---

### 2. 主轴（Main Axis）与交叉轴（Cross Axis）

* **主轴**：默认是水平方向，从左到右。
* **交叉轴**：与主轴垂直，默认是垂直方向，从上到下。

你也可以通过 `flex-direction` 改变主轴方向：

```css
flex-direction: row | row-reverse | column | column-reverse;
```

---

## 二、常用属性

### 1. 父容器属性

| 属性                | 说明          |
| ----------------- | ----------- |
| `flex-direction`  | 主轴方向        |
| `flex-wrap`       | 是否换行        |
| `justify-content` | 主轴对齐方式（水平）  |
| `align-items`     | 交叉轴对齐方式（垂直） |
| `align-content`   | 多行时的交叉轴对齐   |

### 示例：

```css
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}
```

---

### 2. 子元素属性

| 属性           | 说明                                            |
| ------------ | --------------------------------------------- |
| `flex`       | 快捷设置 `flex-grow`, `flex-shrink`, `flex-basis` |
| `align-self` | 单独设置交叉轴对齐                                     |
| `order`      | 改变元素的排列顺序                                     |

```css
.item {
  flex: 1; /* 平均分配剩余空间 */
}
```

---

## 三、Flex 应用场景

### ✅ 常见使用场景：

1. **导航栏水平排列，子项自动间距**：

   ```css
   display: flex;
   justify-content: space-between;
   ```

2. **响应式等高布局**（如三列等高卡片）：

   ```css
   display: flex;
   align-items: stretch;
   ```

3. **元素居中对齐**（经典用途）：

   ```css
   display: flex;
   justify-content: center;
   align-items: center;
   ```

4. **表单左右排列（Label + Input）**：

   ```css
   display: flex;
   flex-direction: row;
   ```

5. **自适应宽度按钮/标签**：

   ```css
   .button {
     flex: 1; /* 占据剩余空间 */
   }
   ```

---

## 四、优缺点对比

| 优点         | 缺点             |
| ---------- | -------------- |
| 简洁、易读      | 一维布局，无法处理网格型需求 |
| 响应式友好      | 复杂布局时略显局限      |
| 子元素自适应空间分配 | 不支持 IE9 以下     |

---

## 五、Flex vs Grid 简要对比

| 特点  | Flex    | Grid    |
| --- | ------- | ------- |
| 维度  | 一维（横或纵） | 二维（横+纵） |
| 场景  | 内容驱动    | 布局驱动    |
| 控制力 | 自动对齐更强  | 精准布局更强  |

---

如需更复杂的网格布局（比如卡片墙、相册等），建议使用 **CSS Grid**。
