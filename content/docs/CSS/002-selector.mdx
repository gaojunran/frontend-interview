这是 CSS 面试中非常经典的三个问题，下面我分别详细讲解：

---

### 一、CSS 选择器有哪些？

CSS 选择器用于选择 HTML 元素，以便应用样式。常见的选择器类型如下：

#### 1. **基础选择器**

| 类型     | 示例         | 说明                   |
| ------ | ---------- | -------------------- |
| 元素选择器  | `div`, `p` | 选中所有该标签的元素           |
| 类选择器   | `.box`     | 选中 class 为 `box` 的元素 |
| ID 选择器 | `#header`  | 选中 id 为 `header` 的元素 |
| 通配符选择器 | `*`        | 选中所有元素               |

#### 2. **组合选择器**

| 类型      | 示例        | 说明            |
| ------- | --------- | ------------- |
| 后代选择器   | `div p`   | div 内所有 p 元素  |
| 子选择器    | `ul > li` | 直接子元素 li      |
| 相邻兄弟选择器 | `h1 + p`  | h1 后的第一个 p 元素 |
| 一般兄弟选择器 | `h1 ~ p`  | h1 后所有兄弟 p 元素 |

#### 3. **属性选择器**

| 示例              | 说明               |
| --------------- | ---------------- |
| `[type]`        | 选中有 `type` 属性的元素 |
| `[type="text"]` | 属性值为 `text`      |
| `[type^="te"]`  | 属性值以 `te` 开头     |
| `[type$="xt"]`  | 属性值以 `xt` 结尾     |
| `[type*="e"]`   | 属性值中包含 `e`       |

#### 4. **伪类选择器**

| 示例              | 说明               |
| --------------- | ---------------- |
| `:hover`        | 鼠标悬浮             |
| `:focus`        | 获取焦点             |
| `:first-child`  | 第一个子元素           |
| `:last-child`   | 最后一个子元素          |
| `:nth-child(2)` | 第2个子元素           |
| `:not(.box)`    | 不含有 `.box` 类名的元素 |

#### 5. **伪元素选择器**

| 示例               | 说明          |
| ---------------- | ----------- |
| `::before`       | 元素内容前插入虚拟内容 |
| `::after`        | 元素内容后插入虚拟内容 |
| `::first-letter` | 选中文本的第一个字母  |
| `::first-line`   | 选中文本的第一行    |

---

### 二、CSS 选择器优先级（Specificity）

CSS 优先级通过一个四位数来衡量（a, b, c, d）：

| 类型         | 表示 | 说明                   |
| ---------- | -- | -------------------- |
| 行内样式       | a  | 如：`style="..."`，值为 1 |
| ID 选择器     | b  | 每个 ID 加 1            |
| 类、属性、伪类选择器 | c  | 每个类名、属性、伪类加 1        |
| 元素、伪元素选择器  | d  | 每个元素名、伪元素加 1         |

#### 计算示例：

```css
div .box #header { color: red; }
```

* `div`：元素选择器 → d=1
* `.box`：类选择器 → c=1
* `#header`：ID 选择器 → b=1
  → specificity = (0, 1, 1, 1)

优先级越高的规则会覆盖其他优先级低的规则；优先级相同则后声明的生效。

### 三、哪些属性可以继承？

并非所有 CSS 属性都会继承，主要是**文本和表格相关属性**可以继承，其他大多数布局、盒模型属性不会继承。

#### ✅ 可继承的常见属性

* 文本相关：

  * `color`
  * `font`, `font-size`, `font-family`, `font-weight`, `font-style`
  * `letter-spacing`, `word-spacing`, `line-height`
  * `text-align`, `text-indent`, `visibility`, `white-space`
* 表格相关：

  * `border-collapse`, `border-spacing`
  * `caption-side`, `empty-cells`

#### ❌ 不继承的常见属性（需手动设置或用 `inherit`）

* 布局相关：`margin`, `padding`, `border`, `display`, `position`
* 盒模型：`width`, `height`, `box-sizing`
* 浮动、定位：`float`, `z-index`, `top`, `left` 等
* 背景、边框：`background-*`, `border-*`

#### 补充：

* 所有属性都可以使用 `inherit` 强制继承父级值。

  ```css
  p {
    margin: inherit; /* 强制继承 */
  }
  ```
