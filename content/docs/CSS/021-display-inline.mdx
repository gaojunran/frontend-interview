---
title: "display: inline 的元素设置 margin 和 padding 会生效吗? 标记为 inline 的元素还有哪些表现？"
---

`display: inline` 的元素设置 `margin` 和 `padding` 会**部分生效**，具体如下：

---

### ✅ `inline` 元素的 `margin` 和 `padding` 表现：

#### ✅ **水平方向：生效**

* `margin-left` / `margin-right` 有效
* `padding-left` / `padding-right` 有效

#### 🚫 **垂直方向：部分生效**

* `margin-top` / `margin-bottom` **不会影响布局**，虽然在开发者工具中能看到，但不会真正占据空间
* `padding-top` / `padding-bottom` 会在视觉上生效，但**不会推动周围元素的位置**，只影响背景颜色等视觉效果

---

### ✅ `inline` 元素的其他表现：

1. **不支持设置宽度和高度**

   * 设置 `width` / `height` 无效，元素宽度由内容撑开

2. **换行特性**

   * `inline` 元素不会换行，多个 inline 元素会排在一行，直到排不下才换行

3. **盒模型**

   * `inline` 元素的盒模型只在水平方向参与布局，垂直方向由字体行高（`line-height`）和字体大小决定

4. **典型的 inline 元素**

   * `<span>`、`<a>`、`<strong>`、`<b>`、`<i>`、`<em>`、`<img>`（虽然 `<img>` 可以设置尺寸，但它本质上是 `inline`）

---

### ✅ 如果你希望 `inline` 元素既能排在一行，又能设置尺寸或垂直 margin：

可以用：

* `display: inline-block`
  👉 同时具有 inline 和 block 的特性：可以设置宽高、margin/padding 全部生效，又不会换行。

---

### ✅ 小结

| 属性/行为             | `inline`        | `inline-block` |
| ----------------- | --------------- | -------------- |
| 设置宽高              | ❌ 无效            | ✅ 有效           |
| 水平 margin/padding | ✅ 有效            | ✅ 有效           |
| 垂直 margin         | ❌ 不占空间          | ✅ 有效           |
| 垂直 padding        | ✅ 有视觉效果（但不影响布局） | ✅ 有效           |
| 换行行为              | 不换行             | 不换行            |
