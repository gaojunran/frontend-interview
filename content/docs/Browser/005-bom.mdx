BOM（Browser Object Model，浏览器对象模型）是浏览器提供的一套操作浏览器窗口和环境的 API，它是 JavaScript 与浏览器环境交互的接口。
相对于 DOM（Document Object Model）是操作网页内容的接口，**BOM 更关注浏览器本身的功能**，比如控制窗口、获取浏览器信息、跳转页面、管理历史记录等。

---

## 一、BOM 的组成

BOM 并没有一个明确的标准，但主流浏览器都实现了一套通用的 BOM 接口，主要包括以下几个常见对象：

---

## 二、常见 BOM 对象及功能详解

### 1. `window` 对象（核心）

* 是 BOM 的顶级对象，代表浏览器的窗口。
* 所有 BOM、DOM 对象都是它的属性。
* 也是 JavaScript 全局作用域的顶层对象（全局变量会挂在 window 上）。

#### 常用 API：

```js
// 弹窗相关
window.alert('提示');
window.confirm('确定吗？');
window.prompt('请输入：');

// 定时器
let timer = window.setTimeout(fn, 1000);
window.clearTimeout(timer);
let interval = window.setInterval(fn, 1000);
window.clearInterval(interval);

// 大小、位置
window.innerWidth / innerHeight; // 可视区域宽高
window.open('https://example.com'); // 打开新窗口
```

---

### 2. `navigator` 对象（浏览器信息）

* 提供有关浏览器的信息，比如用户代理、平台、是否支持插件等。

#### 常用属性：

```js
navigator.userAgent;      // 用户代理字符串
navigator.language;       // 浏览器语言
navigator.platform;       // 操作系统平台
navigator.onLine;         // 是否在线
navigator.geolocation;    // 地理定位 API
```

---

### 3. `location` 对象（地址栏信息）

* 表示当前页面的 URL，可读可写。
* 用于获取或跳转页面。

#### 常用属性/方法：

```js
location.href;        // 获取或设置 URL
location.protocol;    // 协议（http:）
location.hostname;    // 主机名
location.port;        // 端口
location.pathname;    // 路径
location.search;      // 查询参数（?key=value）
location.hash;        // 哈希（#section）

location.assign('url');    // 跳转（可回退）
location.replace('url');   // 替换（不可回退）
location.reload();         // 刷新页面
```

---

### 4. `history` 对象（浏览历史）

* 用于操作用户的浏览历史。

#### 常用方法：

```js
history.back();     // 后退
history.forward();  // 前进
history.go(-1);     // 任意跳转，0 为刷新

history.pushState() 和 replaceState() // 用于 SPA 路由控制
```

---

### 5. `screen` 对象（屏幕信息）

* 提供关于用户屏幕的信息（分辨率等）。

#### 常用属性：

```js
screen.width;      // 屏幕宽度
screen.height;     // 屏幕高度
screen.availWidth; // 可用宽度
screen.availHeight;// 可用高度
```

---

### 6. `console` 对象（调试工具）

虽然不属于 BOM 的核心组成，但常被使用：

```js
console.log(), console.error(), console.table(), console.time()
```

---

## 三、BOM 的应用场景

* 弹窗交互：`alert/confirm/prompt`
* 页面跳转与刷新：`location`
* 用户身份识别或设备判断：`navigator.userAgent`
* 页面访问轨迹控制：`history`
* 浏览器兼容判断
* 前端 SPA 路由：`history.pushState`、`popstate` 事件

---

## 四、总结

| 对象          | 作用           |
| ----------- | ------------ |
| `window`    | 顶级对象，控制窗口及全局 |
| `navigator` | 浏览器信息        |
| `location`  | 控制/获取 URL 地址 |
| `history`   | 页面跳转控制       |
| `screen`    | 屏幕相关信息       |

BOM 是 JS 操作浏览器环境不可或缺的一部分，配合 DOM 和事件机制，构成了完整的浏览器端开发能力。
