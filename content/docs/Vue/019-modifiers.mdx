好的，以下是关于 Vue3 中常用修饰符及其应用场景的回答，适用于面试场景：

---

在 Vue3 中，常用的**事件修饰符**和**v-model 修饰符**依然保留，并且功能更加强大和明确。

---

### 一、常用的修饰符分类

#### ✅ 1. **事件修饰符**

| 修饰符        | 含义与应用场景                                      |
| ---------- | -------------------------------------------- |
| `.stop`    | 阻止事件冒泡。例如：点击按钮时不希望触发父级的点击事件。                 |
| `.prevent` | 阻止默认事件。例如：阻止表单提交后刷新页面。                       |
| `.capture` | 使用事件的捕获模式。适用于提前拦截事件。                         |
| `.self`    | 只有事件从自身触发时才执行回调，适合避免子元素触发。                   |
| `.once`    | 事件只触发一次，适合初始化类操作。                            |
| `.passive` | 提升滚动性能，监听时不阻止默认行为，常用于`touchstart`、`wheel`事件。 |

📌 示例：

```vue
<button @click.stop="handleClick">点我不会冒泡</button>
<form @submit.prevent="onSubmit">...</form>
<div @click.self="onBackgroundClick">模态背景</div>
```

---

#### ✅ 2. **v-model 修饰符**

| 修饰符       | 含义与应用场景                                      |
| --------- | -------------------------------------------- |
| `.lazy`   | 在 `change` 事件后才更新数据，而不是 `input`，适合频繁输入时节省性能。 |
| `.number` | 自动将输入值转为 `Number` 类型，适合数字输入框。                |
| `.trim`   | 自动去除首尾空格，适合表单提交前清洗数据。                        |

📌 示例：

```vue
<input v-model.lazy="name" />
<input v-model.number="age" />
<input v-model.trim="username" />
```

---

#### ✅ 3. **键盘修饰符**

| 修饰符                | 含义与应用场景        |
| ------------------ | -------------- |
| `.enter`           | 回车键触发，常用于表单提交。 |
| `.esc`             | 取消输入、关闭弹窗。     |
| `.tab`、`.delete` 等 | 可以监听特定按键操作。    |

📌 示例：

```vue
<input @keyup.enter="onEnter" />
```

---

### 二、应用场景总结

* 表单防抖提交：`@submit.prevent`
* 模态遮罩点击关闭：`@click.self`
* 数字输入处理：`v-model.number`
* 性能优化：`@scroll.passive`
* 限制一次性行为：`@click.once`

---

### 三、Vue3 特点说明

* 修饰符的语义没有变化，但在组合式 API 中也可以配合 `defineEmits`、`v-model` 多参数的写法结合使用，增强了灵活性。
* 在 `<script setup>` 中，模板中使用修饰符同样有效，语法更简洁。
