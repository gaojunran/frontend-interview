---
title: JavaScript 字符串的常用方法有哪些？
---

JavaScript 中字符串（String）是非常常用的数据类型，常用的方法主要可以分为以下几类：查找、提取、操作、转换、检查等。以下是常见的字符串方法：

---

### 🔍 一、查找类方法

| 方法              | 功能说明            | 示例                                  |
| --------------- | --------------- | ----------------------------------- |
| `indexOf()`     | 查找子字符串第一次出现的位置  | `'hello'.indexOf('l')` → `2`        |
| `lastIndexOf()` | 查找子字符串最后一次出现的位置 | `'hello'.lastIndexOf('l')` → `3`    |
| `includes()`    | 判断字符串是否包含指定子串   | `'hello'.includes('he')` → `true`   |
| `startsWith()`  | 判断是否以指定子串开头     | `'hello'.startsWith('he')` → `true` |
| `endsWith()`    | 判断是否以指定子串结尾     | `'hello'.endsWith('lo')` → `true`   |

---

### ✂️ 二、提取类方法

| 方法                      | 功能说明                 | 示例                                 |
| ----------------------- | -------------------- | ---------------------------------- |
| `slice(start, end)`     | 提取字符串的一部分，不修改原字符串    | `'hello'.slice(1, 3)` → `'el'`     |
| `substring(start, end)` | 类似 `slice`，但不支持负值    | `'hello'.substring(1, 3)` → `'el'` |
| `substr(start, length)` | 从指定位置提取指定长度的子串（已不推荐） | `'hello'.substr(1, 2)` → `'el'`    |

---

### 🧩 三、操作类方法

| 方法             | 功能说明      | 示例                                           |
| -------------- | --------- | -------------------------------------------- |
| `replace()`    | 替换字符串中的子串 | `'hello'.replace('l', 'x')` → `'hexlo'`      |
| `replaceAll()` | 替换所有匹配的子串 | `'hello'.replaceAll('l', 'x')` → `'hexxo'`   |
| `concat()`     | 拼接字符串     | `'hello'.concat(' world')` → `'hello world'` |
| `repeat(n)`    | 重复字符串 n 次 | `'ha'.repeat(3)` → `'hahaha'`                |

---

### 🔁 四、转换类方法

| 方法                          | 功能说明      | 示例                                   |
| --------------------------- | --------- | ------------------------------------ |
| `toUpperCase()`             | 转为大写      | `'hello'.toUpperCase()` → `'HELLO'`  |
| `toLowerCase()`             | 转为小写      | `'HELLO'.toLowerCase()` → `'hello'`  |
| `trim()`                    | 去除首尾空格    | `' hello '.trim()` → `'hello'`       |
| `trimStart()` / `trimEnd()` | 去除开头或结尾空格 | `' hello '.trimStart()` → `'hello '` |

---

### 🔄 五、转换为数组

| 方法                 | 功能说明          | 示例                                       |
| ------------------ | ------------- | ---------------------------------------- |
| `split(separator)` | 按分隔符将字符串分割成数组 | `'a,b,c'.split(',')` → `['a', 'b', 'c']` |
