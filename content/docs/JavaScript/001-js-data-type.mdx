---
title: 说说JavaScript中的数据类型？存储上的差别？
---

在JavaScript中，数据类型主要分为**基本类型（Primitive Types）**和**引用类型（Reference Types）**两大类，它们在**存储方式上存在本质差别**。下面是详细说明：

---

### ✅ JavaScript 中的主要数据类型：

#### **1. 基本类型（Primitive Types）**

这些类型的值是**不可变的**，**直接存储在栈（stack）内存中**。

* `String`：字符串，如 `"hello"`
* `Number`：数字，如 `42`
* `Boolean`：布尔值，`true` 或 `false`
* `Undefined`：变量声明了但未赋值
* `Null`：表示空或无值
* `BigInt`：表示超大整数（ES2020 引入）
* `Symbol`：唯一值（ES6 引入）

#### **2. 引用类型（Reference Types）**

这些类型的值是**可变的**，**实际存储在堆（heap）内存中**，变量在栈中保存的是对堆中对象的引用（地址）。

* `Object`：对象
* `Array`：数组
* `Function`：函数（本质也是对象）
* `Date`、`RegExp` 等内建对象

---

### 🧠 存储上的差别：

| 类型分类 | 存储位置      | 数据特性        | 举例                                             |
| ---- | --------- | ----------- | ---------------------------------------------- |
| 基本类型 | 栈内存       | 值不可变，直接赋值复制 | `let a = 5; let b = a;`（两者互不影响）                |
| 引用类型 | 堆内存 + 栈指针 | 存储地址，引用传递   | `let obj1 = {x:1}; let obj2 = obj1;`（两者指向同一地址） |
