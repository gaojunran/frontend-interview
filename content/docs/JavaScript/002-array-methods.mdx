---
title: JavaScript 数组的常用方法有哪些？
---

JavaScript 数组的常用方法非常多，以下是一些在开发中经常用到的方法，按功能分类整理如下：

---

### ✅ **添加/删除元素**

| 方法                                     | 说明                         |
| -------------------------------------- | -------------------------- |
| `push()`                               | 在数组末尾添加一个或多个元素，返回新长度       |
| `pop()`                                | 删除数组最后一个元素，返回被删除的元素        |
| `unshift()`                            | 在数组开头添加一个或多个元素，返回新长度       |
| `shift()`                              | 删除数组第一个元素，返回被删除的元素         |
| `splice(start, deleteCount, ...items)` | 从某个索引开始，删除指定数量的元素，并可以添加新元素 |

---

### 🔍 **查找/判断元素**

| 方法                    | 说明                        |
| --------------------- | ------------------------- |
| `includes(value)`     | 判断数组是否包含某个值，返回布尔值         |
| `indexOf(value)`      | 返回第一个匹配值的索引，找不到返回 -1      |
| `lastIndexOf(value)`  | 返回最后一个匹配值的索引              |
| `find(callback)`      | 找出第一个符合条件的元素，返回该元素        |
| `findIndex(callback)` | 找出第一个符合条件的元素的索引           |
| `every(callback)`     | 所有元素都满足条件返回 true，否则 false |
| `some(callback)`      | 只要有一个元素满足条件就返回 true       |

---

### 🔄 **遍历/转换**

| 方法                               | 说明                   |
| -------------------------------- | -------------------- |
| `forEach(callback)`              | 遍历数组，不返回新数组          |
| `map(callback)`                  | 遍历数组，返回一个新数组（元素做了映射） |
| `filter(callback)`               | 过滤符合条件的元素，返回新数组      |
| `reduce(callback, initialValue)` | 将数组归约成一个值（如求和）       |
| `flat(depth)`                    | 将嵌套数组“拉平”成一维数组       |
| `flatMap(callback)`              | 先映射后扁平化数组            |

---

### 📋 **排序/填充/复制**

| 方法                               | 说明              |
| -------------------------------- | --------------- |
| `sort(compareFunction)`          | 对数组进行排序         |
| `reverse()`                      | 反转数组元素顺序        |
| `fill(value, start, end)`        | 用固定值填充数组的部分元素   |
| `copyWithin(target, start, end)` | 在数组内部复制一部分到其他位置 |

---

### 🔗 **连接/切片**

| 方法                  | 说明                 |
| ------------------- | ------------------ |
| `concat(...arrays)` | 合并一个或多个数组，返回新数组    |
| `join(separator)`   | 将数组转换为字符串，用指定分隔符连接 |
| `slice(start, end)` | 返回数组的一部分，不改变原数组    |
